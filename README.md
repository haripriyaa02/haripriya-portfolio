# Haripriya K — Portfolio

An interactive, chatbot-style personal portfolio built with React + Vite. Instead of scrolling through static sections, visitors can ask the chatbot questions like "show projects" or "what are your skills" and get real-time responses — paired with a scroll-driven storytelling journey and subtle interactive details throughout.

🔗 **Live site:** https://haripriya-k.netlify.app/

## Features

- 🤖 Chatbot-style hero section with keyword-based responses and typing animation
- 🎤 Voice input support (Web Speech API)
- 📖 Scroll-driven storytelling journey through education, internship, and projects
- 🌗 Light / dark mode toggle
- 🔊 Subtle sound effects (Web Audio API)
- 🖱️ Custom cursor, 3D card tilt on hover, and scroll-triggered reveal animations
- 📄 Resume download button
- 📱 Fully responsive

## Tech Stack

React · Vite · Web Speech API · Web Audio API · CSS

## Run Locally

```bash
git clone https://github.com/haripriyaa02/haripriya-portfolio.git
cd haripriya-portfolio
npm install
npm run dev
```
